## 操作方法
```
$ git clone git@github.com:YuriAkita/vending_program.git
$ cd vending_project
$ irb
$ require 'フルパスコピー'
# ユーザー操作をする場合(0.やめる 1.お金を入れる 2.商品を買う)
$ user = User.new
$ user.index
# 管理者ユーザーを操作する場合(0.やめる 1.商品補充 2.商品追加 3.商品撤去 4.在庫確認 5.売上確認)
$ admin = Admin.new
$ admin.admin_index
```
